<h1 class="h1">Data Structures in R Programming</h1>
<p>Data structures are very important to understand. Data structure are the objects which we will manipulate in our day-to-day basis in R. Dealing with object conversions is the most common sources of despairs for beginners. We can say that everything in R is an object. </p>
<p>R has many data structures, which include:</p>
<img src="imgs/r-programming-data-structures.png" class="imageright" alt="R Programming data structures">
<ol class="points">
<li>Atomic vector</li>
<li>List</li>
<li>Array</li>
<li>Matrices</li>
<li>Data Frame</li>
<li>Factors</li>
</ol>
<hr>
<h3 class="h3">Vectors</h3>
<p>A vector is the basic data structure in R, or we can say vectors are the most basic R data objects. There are six types of atomic vectors such as logical, integer, character, double, and raw. <strong>"A vector is a collection of elements which is most commonly of mode character, integer, logical or numeric"</strong> A vector can be one of the following two types:</p>
<ol class="points">
<li>Atomic vector</li>
<li>Lists</li>
</ol>
<h3 class="h3">List</h3>
<p>In R, <strong>the list</strong> is the container. Unlike an atomic vector, the list is not restricted to be a single mode. A list contains a mixture of data types.
