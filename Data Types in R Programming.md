<h1 class="h1">Data Types in R Programming</h1>
<p>In programming languages, we need to use various variables to store various information. Variables are the reserved memory location to store values. As we create a variable in our program, some space is reserved in memory.</p>
<p>In R, there are several data types such as integer, string, etc. The operating system allocates memory based on the data type of the variable and decides what can be stored in the reserved memory.</p>
<p>There are the following data types which are used in R programming:</p>

<td>True, False</td>
<td>It is a special data type for data with only two possible values which can be construed as true/false.</td>
</tr>
<tr>
<td><strong>Numeric</strong></td>
<td>12,32,112,5432</td>
<td>Decimal value is called numeric in R, and it is the default computational data type.</td>

<tr>
<td><strong>Integer</strong></td>
<td>3L, 66L, 2346L</td>
<td>Here, L tells R to store the value as an integer,</td>

<tr>
<td><strong>Complex</strong></td>
<td>Z=1+2i, t=7+3i</td>
<td>A complex value in R is defined as the pure imaginary value i.</td>
